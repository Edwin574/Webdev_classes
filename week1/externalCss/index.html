Difference between website and internet.

Internet

An internet is a global interconnection of computers.This enables shairing of information and data transfer over the internet through guided transfer protocols.


Website

This is a collection of information found on the internet.It allows documents to be linked to one another via hypertext links. The web uses protocol called Hypertext Transfer Protocol (HTTP).Examples of websites include:

  1. www.google.com
  2. www.amazom.com
  3. www.Facebook.com
  4. www.alibaba.com

Difference between Server and Client computers

Server

This is a computer that allows information to be retrieved remotely upon request.Server softwares allows computers to communicate with other computers.Examples of servers:

  • Apache
  • Microsoft Internet Information Sevices

Client

This is a computer that sends information to the server computer.It also sends sequests to the server.Client computers can be interconnected directly or indirectly.
